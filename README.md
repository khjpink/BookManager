### BookManager   
주요 언어: C#   
프레임워크/플랫폼: .NET Framework   
UI 기술: Windows Forms (WinForms)   
개발 환경: Visual Studio   
      
### 기능   
도서 관리 (CRUD)	도서 정보 추가, 조회, 수정, 삭제   
회원 관리 (CRUD)	회원 정보 추가, 조회, 수정, 삭제   
XML 파일로 자동 저장
